Projeto com front-end em Angular, api em Node e banco de dados em Sqlite

Versões
Angular 16.1.3
Node 18.16.1


Ao baixar o contabilizar-node-sqlite.zip, descompactar, executar na pasta principal o "npm install" para instalação das dependências do projeto Angular, depois entrar na pasta contabilizar-api-node-sqlite e também executar "npm install"
para instalação das dependências do projeto node.

Para testar o projeto, iniciar a api na pasta contabilizar-api-node-sqlite com o comando "node index.js" e depois na
pasta principal efetuar o comando "ng s -o" vai abrir o navegador com a pagina em execução.
